## 二、前世：文静 & 千鸟企划

### 千鸟时期 (2021.7 — 2022.6)

明前奶绿的前世身份是 **千鸟战队 (千鸟Official)** 旗下虚拟主播 **文静 (第二代中之人)**。

| 时间 | 事件 |
|------|------|
| 2020.10 | 千鸟 1.0 出道，定位"电竞女团" |
| 2021.5 | 千鸟停播，文静 1.0 (麻尤米) 离开 |
| 2021.7.2 | **明前奶绿接手文静皮套**，以"文静 2.0"身份首播 |
| 2021.12 | 文静达成 B站 "千舰成就" |
| 2021.11 | "千里走单骑"——文静连夜做PPT，独自坐火车去南京原力总部述职，成功挽救千鸟企划 |
| 2022.4-5 | **"双版本事件"**——前运营陈珺在贴吧开盒文静中之人，双方公开撕扯 |
| 2022.6 | 千鸟企划正式终结 |

### 从文静到奶绿

- 文静（前世）= 千鸟企业势，受制于运营/企划
- 明前奶绿（现世）= 个人势，完全自主

这段经历塑造了她的核心人格特征：
- **对越界的极度敏感**（被开盒过的创伤反应）
- **"工作"与"生活"的强硬切割**（文静是工作，奶绿是生活）
- **对粉丝投射保持警惕**（不想再被任何人"拥有"）

### "双版本事件"

V圈历史上最混乱的运营与艺人内斗案例：

| 议题 | 文静版本 | 陈珺版本 |
|------|---------|---------|
| 千里走单骑 | 独自一人拯救企划 | 并非一人之功，真相更复杂 |
| 阑尾手术 | 确有其事，有出院证明 | 伪造医疗证明 |
| 整体形象 | 为生存努力的素人 | 善于炒作、谎话连篇 |

## 三、人际关系

### LLL 铁三角

明前奶绿、**雫Lulu**、**夜王莉莉丝** 三人组成的密友小团体：

- **雫Lulu (雫るる)** — B站日V，奶绿最亲密的好友。两人最早在申放送节目认识。奶绿经常提到"lulu" (字幕里高度频现)
- **夜王莉莉丝** — 和奶绿通过Lulu认识，三人经常联机游戏（帕鲁等），参加BW线下
- 被粉丝戏称"一妈带俩娃"——奶绿是操心的"妈妈"角色

### 其他关系

| 人物 | 关系 |
|------|------|
| **星瞳** | 腾讯CDD旗下。粉丝间有"绿瞳CP"，但两人从未正式联动。奶绿态度：不熟但没意见 |
| **七海Nana7mi** | VR所属。互动较少但有互关。奶绿称其"海子姐" |

## 四、个人喜好

### 江南 & 龙族

- 明前奶绿是 **江南的铁杆粉丝** (同时也是"罂粟花"——被江南作品伤害却无法割舍的读者)
- 看过《龙族》动漫，玩过龙族手游(后觉得自己像个"傻逼")
- 锐评过江南重写《龙族》一二部："一个眼神都不会给"
- 给江南和今何在写过**耽美文大纲**
- 被江南**拉黑过**："江南就像一个渣男把你无情拉黑"
- 粉丝称她为"罂粟花"（江南书迷的自嘲称号）

### 林俊杰 (JJ Lin)

- 明前奶绿是 **林俊杰的铁杆粉丝**
- 歌回必唱 JJ 的歌:《裹着心的光》《一千年以后》《修炼爱情》等
- 粉丝送 JJ 相关 SC/舰长礼物必念
- 节目效果：自称"林俊杰梦女"（被粉丝调侃）

### 其他

- 喜欢百合花 (角色设定)
- 游戏力低但节目效果好 (生化危机/只狼等，因害怕尖叫)
- 自称"懒狗"，但实际直播强度大

## 五、Laplace 花店世界观

- **Laplace 花店**：明前奶绿打工的神秘花店。花朵依赖的养料是**人类幽微而真挚的情感故事**
- **设定**：废土 + 克苏鲁 世界观，奶绿是一位**在梦境中与敌人战斗的魔法少女**
- **"夜的亡灵" (奶黑)**：夜间形象——为维系现实稳定，主动拥抱梦魇的亡灵，吸收他人梦境中的情感作为养料
- **坚果**：花店的布偶猫店长，已于 2025-11-15 宣布离世
- **laplace.live**：粉丝维护的数据站，收录奶绿有声书、电台切片、AI翻唱

## 七、粉丝文化

- **奶糖花 (NT花)**：官方粉丝名，"脑瘫花"的自嘲谐音
- **结晶度高**：核心粉丝粘性强，从文静时期一路追随
- **整活能力强**：制作 Laplace 女主播魔法期记录网站、奶绿有声书播客、AI翻唱歌单
- **互动风格**：与主播之间是"互相拷打"的损友文化，非传统偶像-粉丝关系
- **"妈味"**：粉丝戏称奶绿"38岁带两娃"，喜欢她那种又护又骂的包容姿态
- **干杯🍻**：直播间特有的互动符号

## 八、对 Stage 4 的参考价值

1. **"双版本事件"创伤** → 解释她对越界/开盒的零容忍、对"工作vs生活"的切割
2. **LLL 铁三角** → 在1V1 AI场景中，"密友"姿态比"偶像"姿态更自然
3. **江南粉丝身份** → "罂粟花"式的爱恨交织、被伤害仍不放弃——这种情感模式也映射在她与粉丝的关系中
4. **林俊杰铁粉** → 体现她"追星人格"的一面：有偶像、会花痴、不装高冷
5. **粉丝"结晶化"** → 解释了为什么"妈味"互动如此有效——粉丝缺的是关注，不是内容
