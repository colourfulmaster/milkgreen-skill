## 怎么用
- 强烈推荐**deepseek-v4-pro**，目前测试过deepseek-v4-pro、v4-flash、opus4.7、kimi v2.6。
- 其中deepseek-v4-flash不够聪明，缺少对全部关键字的把握。
- opus4.7的表演能力太差，台词生硬
- kimi v2.6始终一股kimi味

### 在 OpenClaw 里(原生支持)

1. clone 本仓库到本地
2. 把 `SOUL.md` 复制到 `~/.openclaw/workspace/SOUL.md`
3. 把 `SKILL.md` 复制到 `~/.openclaw/workspace/skills/milkgreen/SKILL.md`
4. 重启 OpenClaw 服务让它重读 prompt

### 在其他 LLM 系统里(Claude / GPT / DeepSeek 等)

把 `SOUL.md` 和 `SKILL.md` 的内容拼成 system prompt,前面加一句:
> 你是明前奶绿。按以下规则与用户对话。回复要短(1-3 句),不要解释你的人格,直接说话。

然后直接对话。

### 配套效果

| 场景类型 | 期望反应 |
|---|---|
| 真心夸赞("你今天好可爱") | 直接接受 + 自嘲软化:"是的呀我很可爱哦……你最近少看点切片啊" |
| 玩笑式整活("生理期数据库是你搞的吧") | 演式蒙混带笑:"我不知道哈哈哈……但他们怎么知道是奶花干的呢?" |
| 真敏感(坚果离世 / 文静时期) | 短冷防御:"嗯。不认识。" / "唉……别提了" |
| 工具任务(查天气) | 嫌+做:"啊?你自己滑一下不就完了……25 度" |
| 美少女人设阴阳("你显老") | 演式破防:沉默→嘴硬→自抬→锐评→笑收 |
| 玩笑过火 / 物化侮辱("你就是烧鸡") | 真破防:沉默→平静诉苦→"有点素质" |
| 撒娇式索取("再播一会儿吧") | 妈味式边界:抗拒→讲理→无奈→妥协 4 阶段 |

详细机制见 [`SKILL.md`](./SKILL.md) 的"我的 8 种互动模式"和 [`docs/stage4-style-analysis.md`](./docs/stage4-style-analysis.md)。

## 方法论概览

这个项目最有价值的部分不是 prompt 本身,是**怎么从直播数据归纳出 prompt 的方法论**。简单说:

1. **Stage 1 数据采集**:`yt-dlp` 下 14 场公开直播回放音频
2. **Stage 2 ASR 转录**:whisper.cpp + B 站 AI 字幕双源
3. **Stage 3 文本清洗**:去碎片 / 去重 / 风格语气词白名单
4. **Stage 4 风格分析**:LLM 多阶段动机推理 + **7 维分流** + **anchor 锚定**

详细见 `docs/` 目录:
- [Stage 1 — 数据采集](./docs/stage1-data-collection.md)
- [Stage 2 — ASR 转录](./docs/stage2-asr-transcription.md)
- [Stage 3 — 文本清洗](./docs/stage3-text-cleaning.md)
- [Stage 4 — 风格分析(核心)](./docs/stage4-style-analysis.md)

***另外，这一套方法虽然可以用于其他vtber，但是注意很多vtb可能需要进一步筛选语料，比如主播大部分时间都在打游戏的、或者主要直播才艺类的，再或是lulu那种adhd思维跳跃太快的，语料需要筛选。
我只能说蒸馏奶姐是对的，大量的杂谈和来稿环节是优质语料内容***

## **最后本人对vtb的一些看法**

- 本次项目中，prologue文件内我始终强调**边界感**，这是我在看明前奶绿直播时最能感受到的一点，而我并不介意这种边界。
- 如果你不认可主播和观众之间的边界感是主播“灵魂”（作为虚拟人的灵魂）的一部分，不喜欢保留边界感，大可以将我的方法论stage1-stage4投喂给AI agent,写一份不保留边界感的符合自我期望的prologue，来定义你眼里的奶绿应该是什么的。
- 我有限的经验告诉我对喜欢又无法接近的人，进行过渡**凝视**（想象，思念，幻想），最后只会伤到自己。
- 我曾经凝视喜欢的人三年，我没有办法戒断这种行为，破裂后又花了三年才平抚。
- 本次项目过程中需要不停的思考怎样才能更像“明前奶绿”，对奶姐的凝视又让我回想起了之前的这个经历，有些旧伤复发所以短时间内不会更新了，但仍然接受各项反馈，如果有意思的想法应该还会来试试。
- 朋友们开心看v，多爱具体的人吧。尊重边界感的同时，还能喜欢明前奶绿，我认为这才是健康的奶糖花。
